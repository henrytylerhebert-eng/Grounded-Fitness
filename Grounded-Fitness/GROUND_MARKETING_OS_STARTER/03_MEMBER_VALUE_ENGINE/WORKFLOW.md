# Member Value Engine Workflow

## Weekly Intelligence Run

### 1. CSV Intake

Input: fake/sample Wodify CSV exports.

Output: source inventory, available fields, and missing fields.

If a source is unavailable, mark `Unknown`.

### 2. Member Engagement Analysis

Identify:

- Attendance frequency.
- Attendance decline.
- Missed weeks.
- New member ramp-up.
- Milestones.
- Recognition opportunities.
- At-risk signals.

### 3. Report Generation

Produce:

- Weekly Owner Brief.
- Weekly Coach Brief.
- At-Risk Member Report.
- Member Milestone Report.
- Member of the Month Candidate Report.
- Lead Pipeline Snapshot.

Reports must separate facts from inferred recommendations.

### 4. Draft Communications

Create draft internal member emails from report findings.

Drafts are not sent automatically.

### 5. Human Review

Reviewer checks accuracy, tone, fit, and risk.

### 6. Optional Human Action

After review, a human may choose to act inside Wodify.

Grounded AI does not create, update, delete, or alter Wodify records.

### 7. Measurement Review

Review source data and outcomes if available.

If missing, record `No measurements found`.
